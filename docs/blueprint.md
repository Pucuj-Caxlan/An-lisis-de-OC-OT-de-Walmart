# **App Name**: Análisis de OC/OT de Walmart

## Core Features:

- Ingesta y análisis de datos: Ingesta datos de archivos Excel (.xlsx) y PDF (.pdf), analízalos y almacénalos en Firebase.
- Normalización de datos: Normaliza datos de diferentes formatos en un esquema unificado.
- Análisis de causa raíz: Identifica causas raíz, recurrencias y patrones en datos de OC/OT por formato, país, año, plan y área, mediante agregación y filtrado.
- Cálculo de impacto y tendencias: Calcula el impacto económico de los OC/OT e identifica tendencias a lo largo del tiempo. Explica cada cálculo/KPI con su fórmula, filtros aplicados y registros contribuyentes.
- Detección de anomalías: Detecta valores atípicos y señales de alerta en datos de OC/OT mediante métodos estadísticos.
- Asistente de descripción: Genera descripciones estandarizadas de OC/OT mediante una herramienta de IA generativa basada en los datos proporcionados, extrayendo de los documentos cargados e identificando información y puntos de datos asociados.
- Replicación de paneles: Replique paneles de nivel de vicepresidente con filtros, KPI, datos históricos, listas de los 10 principales y funciones de desglose.

## Style Guidelines:

- Color principal: Azul intenso (#2962FF) para transmitir confianza, estabilidad e información valiosa.
- Color de fondo: Gris claro (#F0F4F7) para garantizar una apariencia limpia y profesional con una legibilidad óptima.
- Color de acento: Naranja (#FF8F00) para resaltar métricas clave, CTA y alertas.
- Texto del cuerpo: 'Inter' (sans-serif) para una mejor legibilidad.
- Fuente del titular: 'Space Grotesk' (sans-serif) para titulares impactantes. Si se prevé un texto más largo, utilice 'Inter' para el cuerpo.
- Utilice iconos claros y basados ​​en datos que se alineen con los tipos de datos y las unidades de negocio.
- Transiciones y animaciones sutiles al filtrar o desglosar los datos.
